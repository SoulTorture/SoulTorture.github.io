使用hexo+github搭建个人博客

三月 18, 2020

搭建个人博客也是属于心血来潮，相比较简书和csdn之类而言，个人博客是自由的，自定义自己喜欢的风格，记录自己的学习路程和心得。另外一方面，有属于自己的博客，也是件酷酷的事情。搜索网上搭建个人博客，大多是Github+Hexo的组合，步骤也都大同小异，但是由于博客文章较旧，在实时运行当中，还是会出现不少的bug。前人栽树后人乘凉，为避免后来者掉进相同的坑，特写下搭建博客的步骤。 我的博客： https://soultorture.github.io/

二、准备
基于Windows系统（开发者最好使用Linux）

安装Git

安装Node.js

git –version

node –version

git中输入命令，有版本号出现则安装成功。

三、搭建流程
3.1、安装 Hexo

npm install -g hexo-cli
安装成功后，创建一个文件夹（列如我的d盘hexo）

接着进入hexo文件夹,输入一下命令

npm install
运行成功后我们可以查询hexo这个文件下，可以发现文件夹下新增了很多文件，如下

找到hexo文件夹下 _config.yml ，这个文件就是网站主题的的配置文件，后面和github page关联和切换主题时，需要使用到。

在执行hexo s 后，会出现一个网址http://localhost:4000/，将其复制（需要注意的是，在cmd中不可用ctrl + c 来复制，Ctrl + C为停止命令）。打开该网址后，可以看到网站的雏形。

四、自定义主题
4.1、创建github仓库

你的用户名+github.io

4.2、发布

接下来需要使用 Git bash 来进行命令的执行，在hexo文件夹中右击鼠标，你会发现有Git bash 选项，点击进入。执行命令：

npm install hexo-deployer-git –save
最后就是发布了，执行以下命令：

hexo g
hexo d

完成后，你就有属于自己的博客了，在github的setting中 可以看到自己的博客地址.
